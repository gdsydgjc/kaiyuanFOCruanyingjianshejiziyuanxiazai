## 主要功能

1. **闭环FOC电机控制**：该设计能够实现高效的闭环FOC电机控制，适用于需要高精度控制的场景。
2. **步进电机高速控制**：除了FOC控制外，该设计还支持步进电机的高速控制，满足不同应用需求。
